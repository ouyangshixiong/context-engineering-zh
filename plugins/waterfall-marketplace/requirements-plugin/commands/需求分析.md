# 需求分析 Command

1. 创建requirements目录
2. 分析用户输入的文字，是否包含technical words，如果包含使用`requirements-plugin:research-agent`进行技术选型，最后输出`requirements/research-report.md`
3. 使用`requirements-plugin:requirements-agent`分析用户需求（如果有`research-report.md`则合并分析需求），输出详细的需求文档`requirements/requirements.md`，使用checkbox列出所有的需求项（默认是 unchecked）。
